#Project Title 
Design and Implementation of a Prototype Medical Collaborative System for Primary Health Care Services in Ibadan North Local Government Area

## Project Description
A prototype medical collaborative system developed for Primary Health Care services in Ibadan North Local Government Area.

## Features
- User Authentication
- Patient Registration
- Medical Records
- Appointments
- Referrals
- Messaging
- Reports

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript

## Installation

1. Install XAMPP
2. Clone the repository
3. Copy project folder into htdocs
4. Import medical_collab_db.sql into phpMyAdmin
5. Start Apache and MySQL
6. Open:
   http://localhost/medical_collab_system
